\index{coco-scripts|textbf}

The \lstinline{coco-script} module provides the facilities to
render non-Standard Average European script systems (henceforth:
nSAE).

Those fonts can only be used upright (\textit{not} italic!), and only
regular or bold font weights are pre-defined.
